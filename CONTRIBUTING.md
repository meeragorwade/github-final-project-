# Contributing Guidelines

Thank you for your interest in contributing to this project!  
All contributions are greatly appreciated and welcomed.

## Contributions We Welcome

We encourage contributions of all kinds, including but not limited to:

### Bug Reports
- Report issues clearly with steps to reproduce
- Include expected and actual behavior
- Provide screenshots or logs if applicable

### Bug Fixes
- Fix existing issues or bugs
- Ensure the solution is simple and readable
- Test changes before submitting

### Documentation Improvements
- Improve README, comments, or documentation
- Fix typos, formatting issues, or unclear explanations
